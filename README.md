#Install 
`npm install --save @mobileaction/ma-ui-kit`<br>
or <br>
`yarn add @mobileaction/ma-ui-kit`

#Local Development
Project setup <br>
`npm install`

Compiles and hot-reloads for development<br>
`npm run storybook:start`

Lints and fixes files <br>
`npm run lint`

Run your unit tests<br>
`npm run test:unit`
